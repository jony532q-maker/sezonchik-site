# СЕЗОНЩИК — финальная версия сайта

Основной домен: https://sezonchik27.ru
Яндекс.Метрика: 110645856

## Что уже настроено
- адаптивная мобильная версия;
- каталог, поиск, фильтры, избранное и сравнение;
- 35 отдельных страниц велосипедов;
- SEO: title, description, canonical, Open Graph и Schema.org;
- robots.txt и sitemap.xml;
- Яндекс.Метрика и цели действий посетителей;
- favicon, PWA-манифест и Service Worker;
- страница 404;
- оптимизированные изображения и ленивая загрузка.

## Как загрузить
Загрузите содержимое этого архива в корень репозитория GitHub Pages, заменив старые файлы. Файл index.html должен находиться в корне репозитория, а не внутри дополнительной папки.

## Где менять товары
Основные данные каталога, цены, характеристики и наличие находятся в начале файла script.js.
