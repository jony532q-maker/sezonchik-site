Как загрузить с телефона:
1. Нажмите «Скачать обновлённые цены».
2. Откройте репозиторий сайта в GitHub через браузер.
3. Откройте файл prices.js → меню ⋯ → Upload files или замените файл.
4. Загрузите скачанный prices.js и нажмите Commit changes.
5. Через 1–3 минуты цены обновятся на сайте.
