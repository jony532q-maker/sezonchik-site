СЕЗОНЩИК 7.2 — ГОТОВАЯ ВЕРСИЯ

В этом ZIP файлы сайта лежат СРАЗУ В КОРНЕ архива.

Как загрузить через GitHub Desktop:
1. Repository → Show in Explorer.
2. Скопируйте ВСЕ файлы из этого архива в открывшуюся папку.
3. Согласитесь на замену файлов.
4. В GitHub Desktop: Commit to main.
5. Затем обязательно: Push origin.
6. Подождите 1–3 минуты и обновите сайт Ctrl+F5.

Как проверить новую версию:
На первом экране под преимуществами должна быть зелёная кнопка «2ГИС — Отзывы, фотографии и маршрут».
Ниже блока «Почему мы» расположен большой раздел с отзывами 2ГИС.
